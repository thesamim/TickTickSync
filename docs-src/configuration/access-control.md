# Access Control

## TickTick Home Server

Select your home server region before logging in:

- **TickTick** — `https://ticktick.com`
- **Dida** — `https://dida365.com`

Changing servers requires re-authentication.

## Login

Two methods are available — see the [Login page](../login.md) for detailed instructions.

!!! note
    The plugin does **not** save your user ID or password.
