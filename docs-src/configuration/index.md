# Configuration

TickTickSync settings are organized into tabs in the plugin settings panel.

| Tab | Description |
|-----|-------------|
| [Access Control](access-control.md) | Home server selection and login |
| [Sync Control](sync-control.md) | Default folder/project, sync limiting, interval, full vault sync |
| [Maintenance](maintenance.md) | Manual sync, check database, debug mode, backup |
| [Task Settings](task-settings.md) | Link display and task-level preferences |
| [Notes](notes.md) | Note synchronization and separator config |
